#  CodeAlpha Data Analytics Internship

## 🚀 Project Overview

This project is part of the **CodeAlpha Data Analytics Internship**, where I worked on real-world datasets to perform data analysis, visualization, and sentiment analysis.

The project demonstrates a complete **data analysis workflow** including:

* Data exploration
* Data visualization
* Text-based sentiment analysis

---

## 📁 Tasks Completed

###  Task 2: Exploratory Data Analysis (EDA)

* Explored dataset structure and data types
* Checked for missing values and duplicates
* Performed statistical analysis
* Identified trends, patterns, and anomalies
* Tested hypotheses (e.g., effect of test preparation on performance)

📌 **Key Insights:**

* Students who completed test preparation scored higher
* Female students performed better in reading and writing
* Math scores show higher variation
* Dataset is balanced across gender

---

### 📈 Task 3: Data Visualization

* Created meaningful visualizations using **Matplotlib & Seaborn**
* Designed charts to clearly represent patterns and relationships

📊 **Visualizations Created:**

* Bar Chart (Gender Distribution)
* Histogram (Score Distribution)
* Boxplot (Performance Comparison)
* Heatmap (Correlation Analysis)
* Pairplot (Feature Relationships)

📌 **Key Insights:**

* Balanced gender distribution ensures unbiased analysis
* Strong correlation between reading and writing scores
* Visualizations reveal performance trends and patterns

---

###  Task 4: Sentiment Analysis

* Performed sentiment classification using **NLP (TextBlob)**
* Classified text into **Positive, Negative, Neutral**
* Visualized sentiment distribution

📊 **Techniques Used:**

* Natural Language Processing (NLP)
* Sentiment classification
* Word Cloud visualization

📌 **Key Insights:**

* Majority of sentiments are positive
* Negative sentiments highlight dissatisfaction
* Useful for understanding public opinion and trends
